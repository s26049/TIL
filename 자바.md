## 1 | 주석(Comend)
코드에 설명을 추가하여 가독성을 높이고 컴파일러가 무시하게 하여 실행에 영향을 주지 않는 기능
 
**comend**
```java
public class comend {
  public static void main(String []args) {
      System.out.println("Hello java"); // Hello java를 출력한다.
 }
}
```
**실행 결과:**

```java
Hello java
```

### 주석의 종류
<ul>
  <li>
    // 한 줄 주석
  </li>
  <li>
    /*
    여러 줄 주석
    */
  </li>
</ul>

## 2 | 변수(Varialbe)
데이터를 저장하기 위해 이름을 붙인 메모리 공간

**variable**
```java
public class variable {
  public static void main(String []args) {
    int a = 20;
    double b = 10.0;
    boolean c = true;
    char d = 'A';
    String e = "Hello java";

    System.out.println(a);
    System.out.println(b);
    System.out.println(c);
    System.out.println(d);
    System.out.println(e);
 }
}
```

**실행 결과:**
```java
20
10.0
true
A
Hello java
```

```int``` : 정수형 데이터 타입 <br> 
```double``` : 실수형 데이터 타입 <br>
```boolean``` : 참(True)과 거짓(False) 두 가지 값만 가지는 논리형 기본 데이터 타입 <br>
```char``` : 문자 하나를 저장하기 위한 데이터 타입 (작은따옴표 사용) <br>
```String``` : 문자열을 다루기 위한 특별한 데이터 타입 (큰따옴표 사용)

### 리터럴(literal)
코드에서 개발자가 직접 적은 고정된 값


### 변수 명명 규칙
<ul>
  <li> 변수 이름에는 영문자(a-z, A-Z), 숫자(0-9), 달러 기호($) 또는 밑줄(_)만 가능하다.
  </li>
  <li>
    변수의 이름은 숫자로 시작할 수 없다. 그러나 변수 이름 안에 숫자를 포함하는 것은 가능하다.
  </li>
  <li>
    자바의 예약어를 변수 이름으로 사용할 수 없다.
  </li>
  <li>
    변수 이름에는 공백이 포함될 수 없다.
  </li>
</ul>

**참고** : 변수 이름은 의미있고, 그 용도를 명확히 설명해야한다.

## 3 | 연산자(Operator)

### 산술 연산자
가장 기본적인 수학 연산

**operator**
```java
public class operator {
  public static void main(String []args) {
    int a = 5;
    int b = 2;

    int sum = a + b;
    System.out.println(sum);

    int dif = a - b;
    System.out.println(dif);

    int mul = a * b;
    System.out.println(mul);

    int div = a / b;
    System.out.println(div);

    int mod = a % b;
    System.out.println(mod);
 }
}
```

**실행 결과:**
```java
7
3
10
2
1
```

```+``` : 더하기(sum) <br>
```-``` : 빼기(dif) <br>
```*``` : 곱하기(mul) <br>
```/``` : 나누기(div) <br>
```%``` : 나머지(mod) <br>

**참고** : 숫자를 0으로 나눌 수 없다. (수학에서 허용하지 않음)

### 증감 연산자
변수의 값을 1씩 증가/감소 시키며 위치에 따라 실행 순서 변화

**operator**
```java
public class operator {
  public static void main(String []args) {
    int a = 0;
    ++a;
    System.out.println(a);

    int b = 1;
    b--;
    System.out.println(b);
 }
}
```

**실행 결과:**
```java
1
0
```

```++a``` (전위) 값을 1 증가시킨 후 연산에 참여시킴 <br>
```b--``` (후위) 값을 1 감소시킨 후 연산에 참여시킴 <br>

### 비교 연산자
두 값을 비교하여 true 또는 false를 반환

**operator**
```java
public class operator {
  public static void main(String []args) {
    int a = 2;
    int b = 3;

    System.out.println(a == b);
    System.out.println(a != b);
    System.out.println(a > b);
    System.out.println(a < b);
    System.out.println(a >= b);
    System.out.println(a <= b);
 }
}
```

**실행 결과:**
```java
false
true
false
true
false
true
```

```==``` : 같다 <br>
```!=``` : 다르다 <br>
```>``` : 오른쪽보다 왼쪽이 크다 <br>
```<``` : 왼쪽보다 오른쪽이 크다 <br>
```>=``` : 오른쪽보다 왼쪽이 크거나 같다 <br>
```<=``` : 왼쪽보다 오른쪽이 크거나 같다 <br>

### 논리 연산자
여러 조건을 조합할 때 사용

**operator**
```java

 }
}
```

**실행 결과:**
```java

true
false
true
true
false
false
true
```

```&&``` : AND 연산자, 모든 조건이 true인 경우에만 true
```||``` : OR 연산자, 조건이 하나라도 true인 경우 true
```!``` : NOT 연산자, true는 false, false는 true

## 4 | 조건문(Condition)

### if문
If문은 특정 조건이 참(true)인지 확인하고 그 조건이 참(true)일 경우 특정 코드 블록을 실행

**condition**
```java
public class condition {
 public static void main(String []args) {
  int age = 20;
  if ( age >= 18) {
     System.out.println("성인입니다.");
  }
  if ( age <= 18) {
     System.out.println("미성년자입니다.");
  }
 }
}
```

**실행 결과:**
```java
성인입니다.
```

**else문을 사용하면 위 코드보다 더 간략하게 바꿀 수 있다.**

**condition**
```java
public class condition {
 public static void main(String []args) {
  int age = 20
  if ( age >= 18 ) {
     System.out.println("성인입니다.");
  } else {
     System.out.println("미성년자입니다.");
  }
 }
}
```
<ul>
 <li>
  기존: 만약 18살 이상이면 "성인입니다." 를 출력하고, 만약 18살 미만이라면 "미성년자입니다." 를 출력해야한다.
 </li>
 <li>
  변경: 만약 18살 이상이면 "성인입니다." 를 출력하고, 그렇지 않으면 "미성년자입니다." 를 출력해야한다.
 </li>
</ul>

**참고** : if문에서 중괄호{}를 생략할 수 있는데, 이 경우엔 if문 다음에 실행할 문장이 하나여야 한다.

### Switch문
특정 변수의 값을 확인하고, 그 값과 일치하는 ```case```로 점프하여 코드를 실행

**condition**
```java
public class condition {
 public static void main(String []args) {
  int grade = 2;
  int coupon;

  switch(grade) {
    case 1 :
        coupon = 1000;
        break;
    case 2 :
        coupon = 2000;
        break;
    case 3 :
        coupon = 3000;
        break;
    default :
        coupon = 500;
  }
System.out.println("발급받은 쿠폰: " + coupon)
 }
}
```

**실행 결과**
```java
발급받은 쿠폰: 2000
```

## 5 | 반복문(Loop)
작업을 반복하여 실행하고 싶을 때 사용

### While문
조건식이 참(true)인 동안 특정 코드 블록을 반복해서 실행

**loop**
```java
public class loop {
 public static void main {
  int count = 0;

  while (count < 3) {
     count = count + 1;
     System.out.println(count);
  }
 }
}
```

**실행 결과:**
```java
1
2
3
```

### Do-While문
While문과 차별되는 점 하나가 있다면, Do-While문은 실행 코드를 한 번 이상 실행한다는 것

**loop**
```java
public class loop {
 public static void main(String []args) {
  int i = 10;
  do {
     System.out.println(i)
     i++
  } While (i < 3);
 }
}
```

**실행 결과:**
```java
10
```

### For문
초기화, 조건 체크, 반복 후의 작업을 한 줄에 처리 가능한 편리함

**loop**
```java
public class loop {
 public static void main(String []args) {
  for(int i = 1; i <= 3; i++) {
     System.out.println(i);
  }
 }
}
```

**실행 결과:**
```java
1
2
3
```

### 중첩 반복문
반복문은 내부에 또 반복문을 만들 수 있다.

**loop**
```java
public class loop {
 public static void main(String []args) {
  for (int i = 0; i < 2; i++) {
     System.out.println("외부 for 시작 i: " + i);
     for (int j = 0; j < 3; j++) {
         System.out.println("-> 내부 for " + i + "-" + j);
     }
     System.out.println("외부 for 종료 i: " + i);
     System.out.println();
  }
 }
}
```

**실행 결과:**
```java
외부 for 시작 i: 0
-> 내부 for 0-0
-> 내부 for 0-1
-> 내부 for 0-2
외부 for 종료 i: 0
외부 for 시작 i : 1
-> 내부 for 1-0
-> 내부 for 1-1
-> 내부 for 1-2
외부 for 종료 i = 1
```

### break;
break를 사용해 반복문을 탈출할 수 있다.

### continue;
continue를 이용해 반복문의 조건식으로 이동할 수 있다.

## 6 | 스코프(Scope)
변수를 사용할 수 있는 범위

### 지역 변수와 스코프

**scope**
```java
public class scope {
 public static void main(String []args) {
  int m = 10;
  if (true) {
     int x = 20;
     System.out.println(m);
     System.out.println(x);
  }
  System.out.println(m);
 }
}
```

**실행 결과:**
```java
10
20
10
```


```int m```
<ul>
 <li>
  main{ } 코드 블록 안에서 선언되었음으로 main{ } 코드 블록이 종료될 때까지 생존 가능한 변수이다.
 </li>
 <li>
  if{ } 블록 내부에도 접근할 수 있음
 </li>
</ul>

```int x```
<ul>
 <li>
  if{ } 블록 안에서 선언되었음으로 if{ }  코드 블록이 종료될 때까지 생존 가능한 변수이다.
 </li>
 <li>
  if{ } 코드 블록이 종료되면 이 변수는 제거됨 (이후 접근하면 컴파일 오류 발생)
 </li>
</ul>

### 스코프 존재 이유
효율적인 메모리 사용과 유지보수하기 좋은 코드를 만들 수 있다.

## 7 | 형변환(Casting)
데이터 타입을 다른 타입으로 바꾸는 것

### 자동 형변환
작은 크기의 데이터 타입을 큰 크기의 데이터 타입으로 저장할 때, 자바 컴파일러가 자동으로 타입을 변환해 주는 것

### 명시적 형변환
큰 크기의 데이터 타입을 작은 크기의 데이터 타입으로 강제로 변환하는 것
