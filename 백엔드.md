## 백엔드란?
백엔드는 **사용자가 직접 보지 않는 서버측 영역** 으로 애플리케이션이나 웹 서비스가 정상적으로 동작하도록 **데이터 처리**, **비즈니스 로직 수행**, **데이터베이스 관리**, **서버 운영** 등을 담당
<br>
사용자가 웹사이트나 앱에서 버튼을 누르거나 정보를 입력하면, 프론트엔드가 요청을 서버로 보내고 백엔드는 이 요청을 받아 필요한 연산을 수행한 뒤 결과 데이터를 다시 프론트엔드에 전달

### 주요 역할

**1. 서버 로직 처리**
사용자의 요청을 받아 필요한 기능을 수행하고 결과를 반환한다. <br>
**2. 데이터베이스 관리**
사용자 정보, 게시글, 상품 정보 등의 데이터를 저장하고 조회, 수정, 삭제하는 작업을 수행한다. <br>
**3. API 제공**
프론트엔드나 다른 서비스가 사용할 수 있도록 API를 설계하고 제공한다. <br>
**4. 보안 및 인증**
로그인, 권한 관리, 데이터 보호 등 보안 관련 기능을 처리한다. <br>
**5. 시스템 안정성 관리**
서버 성능, 트래픽 처리, 오류 관리 등을 통해 서비스가 안정적으로 운영되도록 한다. <br>
